# DevContest

*Developers contest* - проект моих решений олимпиадных и спортивных задач с сайта http://acmp.ru. Проект обновляется по мере наличия свободного времени, которого, как правило, нет вообще (и это я ещё сильно приуменьшаю реальную ситуацию).

Считаю, что мозг, как программирование, математика и любые другие точные науки - подобны мышце. Всё это нужно постоянно тренировать. Иначе - аттрафия, гиподинамия, болото, смерть и (что куда более страшно) - профессиональная деградация :)

Мои контакты есть [на моём сайте](https://www.koshovyi.com/). Я всегда открыт для обсуждения каких-либо решений, оптимальности\правильности\точности ответов или по любым другим запросам.
